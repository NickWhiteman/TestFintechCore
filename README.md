# TestFintechCore

## Запуск приложения

1. Клонируйте репозиторий:

```bash
git clone https://github.com/your-username/TestFintechCore.git
cd TestFintechCore
```

2. Установите зависимости:

```bash
npm i
```

3. Запуск тестов:

```bash
   npm run test
```

4. Запустите приложение:

```bash
npm run start
```
